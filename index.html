Automated Critical Data Notification System Capstone Project

Focus: Workflow Automation (Power Automate/Zapier) & Prompt Engineering

**Goal:** To mitigate the high error rate and delay associated with manually tracking and relaying time-sensitive operational data changes (e.g., flight passenger count updates, meal order adjustments) across multiple internal teams.

Solution Pipeline:

  1. **Trigger:** Scheduled check of a master data source (simulating airline portals).
  2. **Action (Automation):** Power Automate or Zapier detects a change, calculates necessary quantity adjustments for production, and validates the data.
  3. **Content Generation:** **Gemini** drafts an immediate, categorized, and concise summary alert (using a strict system prompt) for the change.
  4. **Delivery:** Alert is posted to the Production Planning and Billing team communication channels (Teams/Slack).

**Impact:** Designed to achieve a significant reduction in critical communication latency, moving from manual, hours-long processing to a near-real-time automated alert system.
